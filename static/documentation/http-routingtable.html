HTTP Routing Table

/browse | /bundle | /comment | /cve | /epss | /search | /sighting | /stats | /system | /user | /vulnerability
 
/browse
GET /browse/ Get the known vendors
GET /browse/{vendor} Get the known products for a vendor
 
/bundle
GET /bundle/ List all bundles
GET /bundle/{bundle_uuid} null
POST /bundle/ null
DELETE /bundle/{bundle_uuid} Endpoint for deleting a bundle
 
/comment
GET /comment/ List all comments
GET /comment/{comment_uuid} null
POST /comment/ Create a new comment related to a security advisory (vulnerability)
DELETE /comment/{comment_uuid} Endpoint for deleting a comment
 
/cve
GET /cve/{vulnerability_id} null
DELETE /cve/{vulnerability_id} Endpoint for deleting a vulnerability
 
/epss
GET /epss/{vulnerability_id} null
 
/search
GET /search/{vendor}/{product} Returns a list of vulnerabilities related to the product
 
/sighting
GET /sighting List all sightings
GET /sighting/ List all sightings
GET /sighting/{sighting_uuid} null
POST /sighting Create a new sighting
POST /sighting/ Create a new sighting
 
/stats
GET /stats/vulnerability/most_commented null
GET /stats/vulnerability/most_sighted null
 
/system
GET /system/configInfo null
GET /system/dbInfo null
GET /system/info null
GET /system/redis_up null
 
/user
GET /user/ List all users
GET /user/me Get information about the currently authenticated user
POST /user/ Create a non-admin user
POST /user/api_key Regenerating the API key of the authenticated user with the current API key
DELETE /user/{user_id} Endpoint for deleting a user
 
/vulnerability
GET /vulnerability/browse/ Get the known vendors
GET /vulnerability/last null
GET /vulnerability/last/{number} null
GET /vulnerability/last/{source} null
GET /vulnerability/last/{source}/{number} null
GET /vulnerability/search/{vendor}/{product} Returns a list of vulnerabilities related to the product
GET /vulnerability/{vulnerability_id} null
POST /vulnerability/ Endpoint for creating and editing vulnerabilities in the local source
DELETE /vulnerability/{vulnerability_id} Endpoint for deleting a vulnerability
